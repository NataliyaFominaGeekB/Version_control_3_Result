# Инструкция для работы с Markdown

## Выделение текста

Чтобы выделить текст курсивом, его нужно обрамить звездочками (*). Напр.: *милота*

Чтобы выделить текст п/ж, его нужно обрамить двойными звёздочками (**). Напр.: **милота**
